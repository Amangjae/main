## 2026-07-15 추가 작업

무엇을 바꿨는지:

- 참석 인원을 선택할 수 있게 바꿨습니다.
- 추천 식당 카드를 PC에서는 2칸, 모바일에서는 1칸으로 보이게 정리했습니다.
- 추천 식당마다 `선택`, `포털 검색`, `지도 보기` 버튼이 나오도록 바꿨습니다.
- `오늘은 가지 않음` 버튼을 따로 넣었습니다.
- 최근 1주일 안에 선택한 식당은 다시 추천하지 않도록 규칙을 넣었습니다.
- Google Apps Script를 연결하면 선택 결과가 Google Sheets에 저장될 수 있도록 예제 파일을 추가했습니다.
- 깨져 보이던 한글 문구를 주요 파일에서 다시 정리했습니다.

왜 바꿨는지:

- 단순히 식당을 보여주는 수준이 아니라, 실제로 오늘 점심을 결정하고 기록까지 남기는 흐름으로 바꾸기 위해서입니다.

어디서 확인하면 되는지:

- GitHub Pages 화면에서 제목 아래에 설정 영역이 보이고, 참석 인원 선택 후 추천 카드가 다시 바뀌면 됩니다.
- 추천 카드 아래 `오늘은 가지 않음` 버튼이 보이면 됩니다.
- `scripts/google_apps_script_example.gs` 파일이 있으면 선택 저장 연결 준비가 된 상태입니다.

다음에는 무엇을 하면 되는지:

- Google Apps Script를 웹앱으로 배포하고 그 주소를 `selection_endpoint`에 넣으면 실제 선택 저장까지 끝납니다.
