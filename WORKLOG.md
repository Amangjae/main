# 작업 메모

이 문서는 "이번에 무엇을 바꿨는지"를 쉬운 말로 적어두는 메모입니다.

## 2026-06-23

### 이번에 한 일

- 이 프로젝트를 "내 컴퓨터에서만 실행하는 프로그램"이 아니라 "GitHub에서 바로 열리는 홈페이지" 방식으로 바꿨습니다.
- 설정 영역이 화면 옆에 있던 부분을 고쳐서, `점심 추천` 제목 바로 아래에 보이도록 옮겼습니다.
- 식당 정보와 추천 결과를 Google Sheets에서 읽고, GitHub Pages 화면에 보여주는 방식으로 정리했습니다.
- 카카오 API로 주변 식당을 찾아서 결과를 자동으로 갱신하는 흐름을 추가했습니다.
- 예전에 쓰던 로컬 서버용 파일과 중복 파일은 정리했습니다.

### 왜 이렇게 바꿨는지

- 사용 목적이 "로컬 테스트"가 아니라 "GitHub를 실제 홈페이지처럼 사용"하는 것이었기 때문입니다.
- 그래서 브라우저에서 바로 열리는 구조로 단순하게 맞췄습니다.

### 어디서 확인하면 되는지

- GitHub Pages 화면에서 `점심 추천` 제목 아래에 설정 영역이 보이면 정상입니다.
- `구글 시트 열기` 버튼이 보이면 시트 연결용 화면도 준비된 상태입니다.

### 다음에 할 일

- GitHub Secrets를 등록해야 합니다.
- Google Sheets에 서비스 계정 공유를 해야 합니다.
- GitHub Actions를 한 번 실행해서 실제 데이터가 들어오는지 확인하면 됩니다.
